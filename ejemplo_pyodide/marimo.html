Notebook exportado como app web

Esta página está preparada para incrustar el HTML generado por marimo en marimo_dist/index.html.

  1. Instala marimo.
  2. Exporta notebooks/demo_marimo.py como HTML WASM.
  3. Sirve esta carpeta por HTTP.
marimo export html-wasm notebooks/demo_marimo.py -o marimo_dist --mode edit
